С их помощью пытались создавать подземные хранилища для хранения газоконденсата, воронки для водохранилищ, пытались строить плотину и отработать технологию поворота рек. Как правило – безуспешно. Ядерные бомбы использовали для тушения горящих газовых скважин. У американцев тоже было множество идей, как использовать ядерное оружие.
