    гвоздики

    Февраль 15, 2012 в blog
    Рембрандт ван Рейн "Портрет женщины с розовой гвоздикой", 1622

    В эпоху Возрождения на языке цветов красные гвоздики были символом чистой любви. Розовые гвоздики символизировали брак. В некоторых странах гвоздика считалась "цветком Бога" и символизировали его глаз, от которого ничто не может укрыться.

    Дирк Якобс "Портрет Помпея Окко", 1531 Андреа Соларио "Человек с розовой гвоздикой", 1495 Ганс Гольбейн Младший "Джордж Саймон", 1533 Ганс Гольбейн Младший "Молодой человек с гвоздикой", 1533 Франц Пурбус Младший "Герцогиня Анна", 1603 (за левым ухом) Неизвестный немецкий художник "Портрет женщины с Орденом Лебедя", 1490 Варфоломей Брейн Старший "Молодой человек с гвоздикой", 1532 Ян ван Эйк "Портрет человека с гвоздикой", 1390-1441 Мастер Легенды о Святой Урсуле "Портрет дамы с гвоздикой", 1480-1500 Ганс Гольбейн Младший, "Портрет купца Гизе", 1532 Йос ван Клеве, "Портрет Максимилиана I" 1459-1519
