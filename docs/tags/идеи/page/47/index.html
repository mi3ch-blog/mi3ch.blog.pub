    Февраль 27, 2010 в blog
    Давайте соберем в одном месте простые вещи, которые работают. Или которые, может быть кто-то не знает Например: Столбик пепла, упавший на скатерть, проще всего убрать аккуратно дотронувшись до него влажным пальцем (палец можно лизнуть) Стенд для фотографирования с отверстиями для лица (рук) называется тантамареска Белое вино можно приготовить из винограда любого цвета (upd. есть несколько исключений) Васаби в наших ресторанах делают из хрена, горчицы, специй и зеленого пищевого красителя
