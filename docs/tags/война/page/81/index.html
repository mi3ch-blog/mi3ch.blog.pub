По мнению Небе, сбрасывание с воздуха на Великобританию фальшивок на сумму в 30 миллиардов фунтов привело бы к финансовому коллапсу и потере британским фунтом статуса мировой валюты. Непосредственному начальнику Небе, Рейнхарду Гейдриху, идея понравилась. Так началась тайная немецкая операция «Бернхард» по массовому изготовлению поддельных банкнот Интересны способы, которыми немцы добивались состаривания британских банкнот.
