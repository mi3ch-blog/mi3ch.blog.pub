    наглядная эволюция

    Октябрь 3, 2016 в blog
    Перед вами гигантская чашка Петри – блюдце с питательной средой для бактерий. По краям белая масса – это бактерии кишечной палочки. Пространство поделено на полосы, где заметно растет концентрация антибиотика (который эти бактерии должен убивать). 0, 1, 10, 100, и по центру — самая мощная концентрация — 1000. Когда бактерии сталкиваются с первой полосой (концентрация 1) – выстраивается заметная стенка. Но потом эта стена "проламывается" — появляется первый мутант, который начинает размножаться дальше и дальше.
