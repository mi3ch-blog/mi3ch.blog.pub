    если слишком долго держать в руках раскаленную докрасна кочергу

    Май 30, 2016 в blog
    Ученые из Ганноверского университета имени Готфрида Лейбница работают над тем, чтобы разработать для роботов искусственную нервную систему и научить роботов чувствовать боль. Для этого они хотят покрыть поверхность роботов сотнями сенсоров, имитирующих кожу и передающих информацию о внешней среде. Например, при столкновении с препятствием робот чувствует «боль» и изменяет направление движения. Эта система сможет, например, не дать роботу случайно покалечить человека. Для человека возможность чувствовать боль является мощнейшей защитной системой.
