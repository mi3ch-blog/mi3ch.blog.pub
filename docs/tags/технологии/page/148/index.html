Основы грандиозной компании-монополиста были заложены девятью купцами из Амстердама, которые в 1594 году объединились для того, чтобы собрать небольшой флот и отправить его в Индию — страну, где в тот момент безраздельно хозяйничали португальцы. Все четыре корабля вернулись с богатой добычей. Их пример был заразителен, и подобные объединения купцов для торговли с Индией и другими странами Азии стали создаваться в огромных количествах.

    История биткоина войдет во все учебники. Ее будут приводить в пример для иллюстрации совершенно противоположных вещей – как человеческой глупости, так и человеческой дальновидности.
