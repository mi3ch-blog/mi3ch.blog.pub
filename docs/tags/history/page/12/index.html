Власти всячески пытались защитить свой символ Рождества — вокруг него возводили двухметровые заборы, его охраняли солдаты и полицейские, козла обливали водой, всюду ставил веб-камеры. Это не помогало — камеры выводили из строя DoS-атакой, а козла снова сжигали. Один раз его сожгли неизвестные, переодетые Санта Клаусами.
