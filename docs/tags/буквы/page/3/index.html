    буквы

    Август 1, 2019 в blog
    В Нидерландах поэты из города Утрехт шесть лет пишут стихотворение потомкам – по одной букве каждую неделю, в течение последних шести лет. Каждую субботу, примерно в полдень, члены гильдии поэтов Утрехта собираются в центре города, в том месте, где заканчивается последняя строка послания, чтобы вырезать следующую букву в камне и установить ее в мощеную булыжником улицу. По мере добавления новых знаков, буквы складываются в слова, затем в предложения и, наконец, появляются стихи.
